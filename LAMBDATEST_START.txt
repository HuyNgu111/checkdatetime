BƯỚC 1: LẤY LAMBDATEST CREDENTIALS (2 phút)
═══════════════════════════════════════════════════════
1. Đăng ký miễn phí tại:
   👉 https://accounts.lambdatest.com/register

2. Đăng nhập và vào Profile:
   👉 https://accounts.lambdatest.com/profile

3. Scroll xuống phần "Access Key"
   → Copy "Username" và "Access Key"


BƯỚC 2: SET CREDENTIALS (30 giây)
═══════════════════════════════════════════════════════
Mở PowerShell và chạy:

$env:LAMBDATEST_USERNAME="your_username_here"
$env:LAMBDATEST_ACCESS_KEY="your_access_key_here"

Thay "your_username_here" và "your_access_key_here" 
bằng credentials bạn vừa copy.


BƯỚC 3: CHẠY TESTS (2 phút)
═══════════════════════════════════════════════════════
Chạy lệnh:

.\run-lambdatest.bat

Script sẽ tự động:
✅ Check Java & Maven
✅ Install Selenium + TestNG dependencies
✅ Compile test code
✅ Chạy tests trên LambdaTest cloud
✅ Hiển thị kết quả
